Config

Get the xfile2.js to your directory and include it to your program. You have to have the fs module installed in your nodejs environment.

The number 2 is its version. If we have next version it gonna be xfile3.js so we know which version we're using.

XFILE COMMANDS

create

Creates a blank file.

write

Writes text to a file:

read

Reads file content:

append

Appends text to the file:

delete

Deletes file:

rename

Renames the file or directory.

exist

Checks existence of the file or directory:

readDir

Lists all file in the directory.

makeDir

Makes a directory. If the directory with that name already existed, xfile will give false.

deleteDir

! Deletes directory included its subdirectory & files.

copy

Copies file to another location. If you not specify the target file name it will show error.

copyDir

Copies directory & its subdirectories and files to other location. It copies also the directory and do it for all subdirectories too. If you don't put the target directory name, it shows the error.

! But if you put the target directory name which is already existed, it will replace that whole directory.

changeModeFile

Changes the file's mode between 000-777. The mode code can be in both string or number. If puts 0 it means 000, so all the user, group and global will be denied.

infoFile

Show file/directory information.
